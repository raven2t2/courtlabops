## Research Details Required

Every lead must have **AT LEAST 2-3 specific details** from research:

**For SA Clubs:**
- Program size (e.g., "60+ district teams")
- Specific initiative (e.g., "Hoops & Her program")
- Sponsor partnership (e.g., "REDARC 3-year renewal")
- Tournament participation (e.g., "Easter Classic host")
- Recent social media signal

**For Affiliates (Micro-influencers):**
- Recent post topic (not generic "your content")
- Engagement metric (e.g., "6.2% engagement rate")
- Audience insight (e.g., "high-intent followers")
- Tone/vibe match (e.g., "authentic coaching voice")
- Content pattern (e.g., "posts 5x/week about player development")

**For Coaches:**
- Pain point they expressed (direct quote or reference)
- Program type (club, private, academy, NBL1)
- Location
- Specific signal (frustration, question, interest)

**For Sponsors:**
- Current sponsorship portfolio
- Budget alignment
- Brand values match
- Recent event participation
